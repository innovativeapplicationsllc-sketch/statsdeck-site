Get started

Add StatsDeck to Claude in two minutes.

No app to download. StatsDeck connects to the Claude you already use, on web or desktop.

1

Open connectors in Claude

In Claude, go to Settings → Connectors (web or desktop). This is where Claude adds new abilities like StatsDeck.

Install screenshot — Settings → Connectors
2

Add StatsDeck

Choose Add custom connector and paste the StatsDeck server URL:

https://app.statsdeck.ai/mcp

Install screenshot — Add custom connector URL
3

Sign in once

Connect with Google in the popup. That's the whole setup — StatsDeck only ever reads stats and your league data; it never posts or changes anything.

Install screenshot — Google sign-in
4

Just ask

Type a question like "Compare my RBs for this week's flex" and Claude will pull live answers through StatsDeck.
